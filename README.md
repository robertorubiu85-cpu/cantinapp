# CantinApp

Gestionale enologico completo per cantina vinicola.

**Accedi:** https://CANTINARUBIU.github.io/cantinapp

## Funzionalità
- Fatturazione elettronica FatturaPA / SDI
- Magazzino vini, lotti e QR nutrizionale (Reg. UE 2021/2117)
- Prima nota e riconciliazione bancaria
- Registro telematico SIAN vitivinicolo
- Integrazione Fatture in Cloud

## Stack
- Frontend: HTML/CSS/JS single-file
- Backend: Supabase (PostgreSQL + Auth + Realtime)
- Hosting: GitHub Pages
